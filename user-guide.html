🚀 The 5-Step Success Path 🔗

1 Import Data in Builder
2 Analyze and generate artifacts
3 Polish Deliverables canvas
4 Present your narrative
5 Execute in Teams and Checklist

🧭 App Structure 🔗

Primary tabs are Clickt, Checklist, and Settings.

Inside Clickt, core modules are Tasks, Builder, and Presentation.

Guest mode includes Builder + Presentation access, while Tasks requires signed-in mode.

Sign In on Device
🛠️

Builder Module 🔗

Data → Analysis → Deliverables

Data

Import CSV/TXT/JSON, select dataset, and track rows/columns/hash markers for reproducibility.

Tip: Use multiple dataset in one space.

Analysis

Create X-Y plots, bar/scatter visuals, tables, filters, derived columns, and grouped aggregates.

Tip: Start from Quick Preset to reduce setup time.

Deliverables

Load artifacts to canvas, adjust layout, then export PNG/DOC/PPT or send directly to Presentation.

Tip: Use Polish All before final export.

🖥️

Presentation Module 🔗

Decks, slide editing, branding, and export

  • Create/edit/reorder slides and decks
  • Control title/subtitle/body/image sections visually
  • Apply template and brand rule settings
  • Preview in-app and export HTML/PDF
Pro Tip: On iPhone/iPad, exports save to your Files app. On Mac, you choose the destination folder.

Teams & Collaboration 🔗

Use Teams to create groups, send membership requests, assign tasks, and run YES/NO decision invites.

OPEN

Task is active and assigned.

COMPLETED

Assignee finished task and status is closed.

YES / NO

Decision invite response for fast team consensus.

🔔 Notification Events 🔗

When cloud functions and device tokens are configured, push notifications can be triggered for team invites, invite responses, task assignment/completion, decision invites, and task-thread messages.

📘 Deep Dive (Optional) 🔗

Additional features currently present in codebase
  • Plan: editable plan document with reset
  • Calendar: provider sync, date-range filters, complete/reopen/delete
  • Data Streams: source registration and validation status
Authentication and access notes

Signed-in users can access team/cloud workflows. Guest users can explore Builder and Presentation without full team features.

🔍 Quick Troubleshooting 🔗

Notification issues?

Verify notification permissions, deployed cloud functions, and active device token registration.

Export failed?

Try PNG first to validate canvas rendering, then retry PDF/DOC/PPT output.

Team actions blocked?

Confirm invite acceptance and creator permissions for creator-only actions like team deletion and task assignment.
